HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer elment contain author, copyright, contact, sitemap, and navigation

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • style can be adjusted via specific tag, infile stylesheet or external stylesheet
  • can create a class in stylesheet with several style features

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds changes to staging area
  • git commit -m: screenshots all the changes
  • git push origin feature/starter-code: pushes changes from local branch to remote branch in github

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Array is a group or related data
  • For loop is commmon method for iterating thru each array element
  • Array first element is 0
  • Use a function when writing set of instructions for a specific task
